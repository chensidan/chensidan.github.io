使用 Hugo&&Github 搭建博客

参考 hugo中文文档、Longli博客、olOWOlo博客、Font Awesome使用教程、hugo官方网址

1. 安装hugo与git

sudo apt-get install hugo
sudo apt-get install git

2. 建立hugo站点

hugo new site mysite #mysite可更换，例如 sidanblog

hugo会自动生成如下文件：
archetypes/ #配置default.md
content/ #储存网站所有文章内容(.md原始文件)
layouts/ #全局样式，优先级高于theme下的layouts文件夹
static/ #静态文件(图片)，优先级高于theme下的static文件夹
data/
config.toml #配置网页文件

3. 创建文章

cd mysite
hugo new about.md #自动生成content/about.md
hugo new content/post/first.md #便于管理，在content/下新建post/，放置内容

新建about.md内容如下：
-–
title: “About”
date: 2022-05-21T15:49:57+08:00
draft: true #draft=true,表示这是一篇草稿,默认不渲染草稿
-–

4. 安装主题

进入hugo的Themes，进行挑选，参考主题Demo进行配置
git clone https://github.com/olOwOlo/hugo-theme-even themes/even

5. 启动hugo

hugo server --buildDrafts --theme=even

浏览器输入http://localhost:1313，预览博客

6. Github部署

申请github账号，新建仓库Repository，名称为用户名(须小写).github.io

baseURL = "chensidan.github.io" #修改config.toml
hugo --theme=even --buildDrafts --baseUrl="https://chensidan.github.io/" #站点根目录下运行

所有静态页面会生成到 public/，并且 pubilc/里所有文件 push 到刚创建的 Repository 的 master 分支

cd public git init
git remote add origin git@github.com:chensidan/chensidan.github.io.git
git add -A
git commit -m "whatever you write"
git push -u origin master

浏览器中输入你的URL，例如:chensidan.github.io,能打开博客即说明部署成功

7. 日常使用

新建markdown博文，后缀名为.md的文件

hugo new content/post/new.md

生成Bash脚本，可打包提交

#!/bin/bash
# 错误时，终止脚本
set -e
# 生成静态页面
hugo --theme=even --buildDrafts --baseUrl="https://chensidan.github.io/"
# 发布
cd public
git add .
git commit -m "new blog added"
git push origin master
cd ../
